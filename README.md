# 📘 The Data Science Interview Book

[![Jupyter Book Badge](https://jupyterbook.org/badge.svg)](https://ml.pages.dev/)
[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Preparing for Data Science and Machine Learning interviews can be overwhelming. This project aims to consolidate essential "recipes"—concise, actionable summaries of key concepts—to help you brush up quickly and effectively.

> **Note:** This book is designed for those already familiar with the topics. It focuses on high-level refreshers rather than deep-dive tutorials.

## 🛠️ Local Development

To build the book locally, ensure you have Python 3.9+ installed.

### 1. Clone the repository
```bash
git clone https://github.com/somnathrakshit/mymlrecipes.git
cd mymlrecipes
```

### 2. Install dependencies
We recommend using `uv` for lightning-fast installation:
```bash
uv pip install -r requirements.txt
```
*Or using standard pip:*
```bash
pip install -r requirements.txt
```

### 3. Build the book
```bash
jupyter-book build ./
```
The output will be available in `_build/html/index.html`.
